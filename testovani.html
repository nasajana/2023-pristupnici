Nástroje pro testování přístupnosti

  • Accessibility Insights
  • Colour Contrast Checker: Rozšíření pro Chrome umožňuje z libovolného místa stránky kapátkem nabrat barvu textu a pozadí a určit jejich kontrastní poměr.
  • Color Contrast Analyzer: Rozšíření pro Chrome analyzuje barvy celé stránky nebo její části a vyznačí místa s nedostatečným kontrastem.
  • Color Contrast Analyser pro Windows a macOS: jednoduchá aplikace umožňuje z libovolného místa obrazovky kapátkem nabrat barvu textu a pozadí a určit jejich kontrastní poměr.
  • WAVE Web Accessibility Tool
  • Deque University for iOS: aplikace pro iOS prakticky demonstrující nejdůležitější zásady pro přístupnost v iOS.
  • Kontrola přístupnosti pro Android: aplikace Googlu pro základní automatickou kontrolu přístupnosti aplikací v systému Android.

Testování přístupnosti s asistivními technologiemi

  • Video: Accessibility Testing with the NVDA Screenreader: Popis instalace, nastavení a používání NVDA pro vidícího testera, který chce NVDA používat k testování přístupnosti.
  • WebAIM: Using NVDA to Evaluate Web Accessibility
  • NVDA Download
  • Focus Highlight NVDA addon: Stažený soubor s příponou .nvda-addon je třeba otevřít při spuštěném NVDA. Doplněk bude nainstalován.
  • WebAIM: Using JAWS to Evaluate Web Accessibility
